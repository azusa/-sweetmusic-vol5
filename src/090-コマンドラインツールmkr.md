# コマンドラインツールmkr

MackerelではコマンドラインツールがOSSとして^[[https://github.com/mackerelio/mkr](https://github.com/mackerelio/mkr)]
提供されています。

インストールは`yum`または`apt`コマンドによって行います。

```
sudo yum install mkr
```

以下に示すのは、Bitbucket Serverのバックアップ処理の運用で
サービスがメンテナンスモードになっている際に、ヘルスチェックの警告が発生しないよう、
MackerlのホストのステータスをMaintenanceとするものです。

```
mkr
```
