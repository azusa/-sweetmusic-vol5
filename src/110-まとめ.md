# まとめ

Mackerelを使用したサーバー監視の手法について述べてきました。

Mackerelは他にもサービス外形監視やAWSを始めとする
パブリッククラウドとの連携機能など、
Webサービスの監視に有用な機能が数多く含まれています。

Mackerelのサービスには、セットアップやプラグイン、API仕様などに
デザインとしてのシンプルさとユーザーフレンドリーなインターフェースが
一貫して特徴づけられています。

そのシンプルさゆえに、パブリッククラウドに限らずとも
様々なWebサービスにおいて、必要最小限な構成から、業務に必要な
監視の仕組みを作り上げることができます。

Mackerelの主な機能は、個人でも無償の範囲で試してみることが可能です。

MackerelはWebサービスのお供となるお手軽な監視サービスとして、
最適なソリューションであるといえます。
